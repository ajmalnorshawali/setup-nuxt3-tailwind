# TUTORIAL 1: INSTALL NUXT

## STEP 1
- Open Terminal and run code
```
npm create nuxt@latest myProject
```

1) Which package manager would you like to use? **npm**
2) Initialize git repository? **No**
3) Would you like to install any of the official modules? **No**

## STEP 2
- Type the following code to open myProject
```
cd myProject
```

- Type the following code to open the Visual Studio Code
```
code .
```

- Type the following code to run myProject
```
npm run dev
```
- Open in browser **http://localhost:3000/**
